Synful uses Spackle for Template parsing.

In this example, we include a JavaScript file to unhide a div element defined in the Template file,
and then replace the substitutions based on the configuration in the template handler within the configured controller.

You can also use templating plugins. Find more information at the Spackle repository.

As an example we have the URLParser and ConfigParser plugins, here is this pages url: {{url /example/template <}} and the value of PRODUCTION in System.yaml is {{conf system.production <}}.

"; echo "And the config value for PRODUCTION in System.yaml is {{conf system.production <}}"; <}}
